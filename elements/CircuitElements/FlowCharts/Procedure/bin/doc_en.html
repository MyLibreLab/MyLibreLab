Flowchart "Procedure/Function" Element

With this Element you can start a Procedure/Function.

A Procedure/Function have on the top the "Start Element" with his name and parameters.

Example : result=Test(a,b,c)

It can start a Procedure/Function from the currect VM or from another VM

To start from the currect VM leave the property "vm-Filename" empty.

To start from another VM fill  the property "vm-Filename" with the name of the VM (without the Extension "vlogic")

Exaple : sub-vm = subvm.vlogic -> subvm
