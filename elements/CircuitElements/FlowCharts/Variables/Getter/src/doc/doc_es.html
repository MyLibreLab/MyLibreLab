ENTREGA DATO (Librería FlowCharts-Variables)

Este bloque permite entregar el valor de una variable a un bloque cualquiera en todo momento de MyOpenLab siempre que sea un dato compatible (bol, dbl, str)

La diferencia con el bloque "Escribir Dato" es que este bloque entrega el dato en todo momento, no cuando la ejecución del diagrama de flujo llega el bloque como ocurre con este bloque "Escribir Dato".
