DESCRIPCIÓN DE FUNCIONES:

 

Char charAt(int index)

Retorna el valor del character ubicado en el índice especificado.

String    concat(String str)

Une el texto especificado al final del texto de entrada.

boolean              contains(CharSequence s)

Retorna Verdadero solo si el texto especificado está contenido en el texto a evaluar.

boolean              endsWith(String suffix)

Prueba si el texto termina con el sufijo especificado.

boolean              equals(Object anObject)

Compares this string to the specified object.

Compara si los textos de entrada son iguales pero se distinguen mayúsculas.

boolean              equalsIgnoreCase(String anotherString)

Compara si los textos de entrada son iguales pero sin distinguir mayúsculas.

byte[]   getBytes()

Codifica el texto en una secuencia de Bytes con el conjunto de caracteres por defecto y retorna ese arreglo de Bytes.

int          indexOf(int ch)

Retorna el indice dentro del texto donde aparece por primera vez el character especificado.

int          indexOf(int ch, int fromIndex)

Retorna el indice dentro del texto donde aparece por primera vez el character especificado iniciando la busqueda desde el índice especificado.

int          indexOf(String str)

Retorna el indice dentro del texto donde aparece por primera vez el texto especificado.

int          indexOf(String str, int fromIndex)

Retorna el indice dentro del texto donde aparece por primera vez el texto especificado iniciando la busqueda desde el índice especificado.

boolean              isEmpty()

Retorna verdadero si, y solo si la longitude el texto es cero.

int          lastIndexOf(int ch)

Retorna el indice dentro del texto donde aparece por última vez el character especificado.

int          lastIndexOf(int ch, int fromIndex)

Retorna el indice dentro del texto donde aparece por última vez el character especificado iniciando la busqueda desde el índice especificado.

int          lastIndexOf(String str)

Retorna el indice dentro del texto donde aparece por última vez el texto especificado.

int          lastIndexOf(String str, int fromIndex)

Retorna el indice dentro del texto donde aparece por última vez el texto especificado iniciando la busqueda desde el índice especificado.

int          length()

Retorna la longitude del texto especificado.

boolean              matches(String regex)

Dice si coincide o no la expression regular especificada con el texto de entrada.

String    replace(char oldChar, char newChar)

Retorna un texto resultante del texto de entrada reemplazando todas las apariciones del character Viejo por el character Nuevo.

String    replace(CharSequence target, CharSequence replacement)

Retorna un texto resultante del texto de entrada reemplazando todas las apariciones del texto Viejo por el texto Nuevo.

String    replaceAll(String regex, String replacement)

Retorna un texto resultante del texto de entrada reemplazando todas las apariciones del formato de texto Viejo por el texto Nuevo.

String    replaceFirst(String regex, String replacement)

Reemplaza el primer subTexto del Texto de entrada que coincide con la expression regular especificada con el texto de reemplazo especificado.

String[]                split(String regex)

Divide el texto según el separador especificado y genera un arreglo de textos.

String[]                split(String regex, int limit)

Divide el texto según el separador especificado y genera un arreglo de textos que no superará en cantidad al valor límite especificado.

boolean              startsWith(String prefix)

Prueba si el texto empieza por el prefijo especificado.

boolean              startsWith(String prefix, int toffset)

Prueba si el texto empieza por el prefijo especificado iniciando desde el índice especificado.

String    substring(int beginIndex)

Retorna un texto que es un subtexto del texto de entrada desde el índice especificado al final del texto.

String    substring(int beginIndex, int endIndex)

Retorna un texto que es un subtexto del texto de entrada desde el índice de inicio especificado hasta el índice final especificado.

String    toLowerCase()

Convierte todos los caracteres en Minúsculas según las regal por defecto del sistema local.

String    toUpperCase()

Convierte todos los caracteres en Mayúsculas según las regal por defecto del sistema local.

String    trim()

Retorna un texto al cual se le renoverán todos los caracteres en blanco.

String    trim_ln()

Retorna un texto al cual se le renoverán todos los caracteres en blanco y los caracteres de retorno de carro y nueva línea.

String    trimAll()

Retorna un texto al cual se le renoverán todos los caracteres en blanco, los caracteres de retorno de carro, nueva línea, tabs y espacios.
