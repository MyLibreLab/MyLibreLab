Operaciones con Cadenas de Caracteres: ASIGNACIÓN DE FORMATO

Este operador permite modificar el formato de una cadena de texto que representa un numero en formato dbl en sui entrada de formato ponemos el patron mediante una cosntate tipo string

Ejemplo: #00.00 este formato saca los numeros con dos cifras enteras y dos decimales

En el ejemplo vemos que el formato es #000.000 tres cifras enteras y tres decimales
