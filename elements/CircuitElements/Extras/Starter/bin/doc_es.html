Funcion EJECUCION DE UNA APLICACION EXTERNA

Esta función permite la ejecución de una aplicación externa.. Para que funcione basta que le indiquemos mediante una cadena (string) el nombre de la aplicación y en otra cadena los parámetros que requiera esta aplicación. La tercera entrada es para ejecutar la orden de ejecución (entrada booleana).

En el ejemplo vemos que se trata de ejecutar la aplicación notepad y el parámetros es el nombre del fichero 1.txt
