EVENTOS DE RATÓN

Esta función es muy útil dentro de MyOpenLab dado que se encarga de recoger los valores e los distintos eventos que

se pueden registra de todas las acciones del ratón y se pueden asociar a variables.

Los valores que devuelve esta fuincion son los siguientes:

onMousePresed = Detecta si se ha presionado una tecla

onMousedReleased = Detecta que que el boton pulsado se ha soltado

onMouseDragged = Detecta el arrastre del raton cuando se mantiene pulsado un botón

onMouseMoved = Detecta que el ratón se esta moviendo

x-coodenate = Marca la cordenada x en donde se encuentra el puntero del raton

y-cordenate = Muestra la coordenada y en donde se encuentra el puntero del ratón

Button = decta si se presionó uno de los botones

Clickcount = Cuenta el numero de clicls que se dan con el boton izquierdo
