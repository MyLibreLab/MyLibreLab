Die Javascript Komponente

Die Javascript Komponente verwendet die in Java eingebaute Javascript Engine.
Vor der Ausführung, wird der JS Code compiliert um die Geschwindigkeit zu verbessern.
Ist eine schnelle Alternative um eigene CircuitPanel Komponenten zu schreiben. Ohne Java SDK!

Wie funktioniert die JS Komponente?

1. Konfiguriere deine Input und Output Pins im Property-Editor
2. Stelle im Menu unter Extras->Optionen->Editoren:Javascript Editor dein lieblings JS Editor.
zb:
  • Notepad++ unter https://notepad-plus-plus.org/
  • Netbeans unter https://netbeans.org/
3. Im Property-Editor dann noch die Taste "..." unter Javascript drücken und deinen JS Code eingeben. Wenn Du deinen JS Code speicherst, wird dieser automatisch in die Komponente übernommen.

InPin2 auf C_DOUBLE (ZoomY)

Was muss ich beachten?

Wenn ein Output Informationen versendet möchte, so muss nach den setzen des Wertes auch die Methode setChanged(true) aufgerufen werden.
Bsp:
out.setvalue("Test");
out.setChanged(true);
