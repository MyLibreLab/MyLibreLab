Der Math-Graph-Generator (MGG) interpretiert einen Mathematischen Ausdruck (Expression) und setzt es in einen 2D Graphen um.

Dabei wird die Variable x für den x-Bereich verwendet.

 

Pins

 

expression

Mathematischer Ausdruck (String)

calc

wenn TRUE wird der Graph generiert

graph

Ausgangsgraph (z.B.: für die Komponente Oscilloscope)

error

gibt den Fehler String aus (z.B.: für die Komponente Message)

 

 

Eigenschaften im Property-Editor

 

minX

minimale Grenze der X-Achse

[-99999999] bis [+99999999]

maxX

maximale Grenze der X-Achse

[-99999999] bis [+99999999]

Step

schrittweite für die Genauigkeit des Graphen

[0.000000001] bis [+1000]

 

 

Beispiele  für einen Ausdruck:

 

Sin(x/20)*20

Sin(x/20)*20*cos(x/2)

80*exp(-((x/40)^2))

x^3*0.0001
