Tarjeta de interface MSE-PCIO-4E4S


Este componente se utiliza para controlar la tarjeta de interface MSE-PCIO-4E4S de Ingeniería de Microsistemas Programados S.L. ( www.msebilbao.com ).

La tarjeta se controla desde un PC, a través del puerto USB, mediante una serie de sencillos comandos ASCII.Estos comandos están completamente encapsulados por el componente MSE-PCIO-4E4S de MyOpenLab, por lo que no es necesario conocer su protocolo.

En www.msebilbao.com puedes encontrar una descripción más detallada de esta tarjeta.

Características básicas de la tarjeta

Hier ein Auszug aus der Webseite:
  • 4 Salidas a relé
  • Corriente máxima de los contactos: 2A a 24VDC; 1A a 125 VAC.
  • 4 entradas digitales opto acopladas de 3.7V a 12 VDC.
  • 4 entradas analógicas de 0 a 5 V.
  • Conexión con el PC a través del puerto USB.
  • Alimentación desde el puerto USB (por defecto).
  • Opcionalmente se puede alimentar mediante una fuente externa de 9 a 12 VDC.
  • Consumo aproximado con todas las entradas y salidas activadas de 250 mA.
  • Dimensiones de la tarjeta de 110 x 76 mm.
  • Disponible Driver de Microchip para WXP, Vista, W7, W8 y W10.

Editor de propiedades

Para este componente sólo se necesitan editar dos propiedades: el puerto COM al que está conectado y la velocidad o tiempo de muestreo (en mS)

El puerto COM

Es el que se usa para conectar la tarjeta de interface MSE-PCIO-4E4S con el puerto USB del PC:
  • Por favor asegúrese de que la tarjeta se conecta al PC antes de comenzar a trabajar con MyOpenLab.
  • Seleccionar el puerto COM apropiado antes de iniciar la máquina virtual (Virtual Machine)

Tiempo de muestreo

El tiempo de muestreo establece el tiempo en mili segundos (mS) en el que la tarjeta de interface MSE-PCIO-4E4S actualiza el estado de sus entradas/salidas.

Valor pequeño = Alta velocidad de muestro y actualización
Valor grande = Baja velocidad de muestreo y actualización
  • NOTA: Para un PC lento se debe seleccionar un tiempo de muestreo mayor o de lo contrario la máquina virtual se puede bloquear.
