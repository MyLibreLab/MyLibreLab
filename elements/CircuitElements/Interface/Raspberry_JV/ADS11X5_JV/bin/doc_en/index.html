ELEMENT NAME: cmdWrite_JV

DEVELOPED BY: Robinson Javier velásquez  EMAIL: javiervelasquez125@gmail.com COUNTRY: Colombia

DESCRIPTION:

This element lets to read Analog to Digital ADS11X5 Modules trought I2C Communication Port. Compatible with ADS1115 (16 bit of resolution) ó ADS1105 (12 bit of resolution).

EThis element is based on ADS11X5 Python Adafruit Library, you will need enable the I2C port on Raspbian Configuration and Install the Library from Linux Terminal.

To install the Library you can use the command sudo pip install adafruit-ads1x15 without quotes.

The default I2C Slave address is 0x48 and the bus id is 1 on raspberry pi model 3B, for models before 2012 year the bus id was 0, you can modify this parameters in the properties list.

You can select the gain of the ADC Module:

#  -   1 = +/-4.096V

#  -   2 = +/-2.048V

#  -   4 = +/-1.024V

#  -   8 = +/-0.512V

#  -  16 = +/-0.256V

#  - 2/3 = +/-6.144V

ELEMENT INPUTS:

1, "Enable_Element" (Boolean):

If rising edge detected it executes the code, if false do not run anything.

 2, "Enable_Debug_Window" (Boolean):

If True  the element launches a pop-up window to show what is happening with the execution of the command. If FALSE do not show the pop-up window.

4, "System_Out" (String):

This output is the response that came from CMD or TERMINAL by the executed command. Depending on the command and system operating this output may not return anything.

For more info about operating system commands and it results please refers to your operating system help.

This indicator shows TRUE if there was error during the execution of the command entered and FALSE if no error.
