DESCRIPCIÓN:

Este elemento permite leer a través de protocolo I2C los módulos de Conversión analógica-digital ADS1115 (16 bit de resolución) ó ADS1105 (12 bit de resolución).

El elemento está basado en la Librería Python de Adafruit, por lo cual antes de usarlo debe activar el puerto I2C en la configuración de Raspbian e instalar la librería.

Para instalar la librería puede usar el comando sudo pip install adafruit-ads1x15 sin comillas en la terminal de Linux.

La dirección del módulo por defecto es 0x48 y el bus del puerto es 1 en raspberry pi model 3B, para modelos anteriores a 2012 el bus era 0 por lo cual en las propiedades del elemento se puede cambiar el bus según la necesidad.

En las propiedades del elemento también se puede seleccionar la ganancia del módulo que según la documentación de la librería modifica los rangos de tensión así:

#  -   1 = +/-4.096V

#  -   2 = +/-2.048V

#  -   4 = +/-1.024V

#  -   8 = +/-0.512V

#  -  16 = +/-0.256V

#  - 2/3 = +/-6.144V

EJEMPLO BÁSICO DE FUNCIONAMIENTO:

Puede encontrar el ejemplo en la carpeta de Ejemplos, ubicada en la ventana de "Proyectos>VirtualMachines> RaspbberryPi> Test_ADS11X5_JV".

ENTRADAS:

1, "Enable_Element" (Booleano):

Si se detecta un flanco ascendente el VM se ejecuta, si es false no hace nada.

 

2, "Enable_Debug_Window" (Booleano):

Si es True muestra una ventana emergente para ver detalles de la ejecución del comando para depurar y corregir el comando si no está funcionando de forma esperada.

SALIDAS:

6, "System_Out" (Cadena String):

Esta salida es la respuesta que llegó del CMD ó TERMINAL por el comando ejecutado. Dependiendo del comando y sistema operativo puede no devolver nada.

Para mayor información de comandos CMD y sus respuestas por favor referirse a la documentación de su sistema operativo.

 7, "Error_Out":

Este indicador muestra True si hubo error en la ejecución del comando ingresado y False si no hubo error.

Por ejemplo si se usa un comando de Linux "ifconfig" en el CMD de windows esto causará un error por la diferencia de sintaxis.
