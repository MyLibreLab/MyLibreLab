modul bus CompuLAB Interface (USB)


Das CompuLab Interface USB Modul ist ein Input / Output Modul der Firma modul bus.

Weitere Infos finden Sie unter : http://www.ak-modul-bus.de/
 

Eingänge:

  • 8 x Digital (TTL 0..5V)
  • 2 x ADC (Analog-Digital-Converter auföäsung 10 Bit (0..1023))  (0..5V)

Ausgänge:

  • 8 x Digital
