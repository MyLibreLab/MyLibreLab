Velleman(R)* K8055 USB Board mit Treiber Version 1.1


Das K8055 USB Board ist ein Input / Output Board der Firma Velleman(R)*.
Es können bis zu 4 Geräte am PC angeschlossen werden.
Dazu muss die Geräteadresse im Property-Editor eingegeben werden.
Standardmäßig ist die Geräteadresse "0".

Geräteadresse:

! Achtung: wenn Pins für die Geräteadresse geändert worden sind muss der USB Stecker raus und rein gesteckt werden damit das gerät unter der neuen Adresse erkannt werden kann.

Eingänge:

  • 5 x Digital
  • 2 x ADC (Analog-Digital-Converter)

Ausgänge:

  • 8 x Digital
  • 2 x PWM (Pulswellenmodulation)

Zusätzlich:

  • 2 Counter die an Input 1 und 2 verbunden sind.
     

Counter Debounce Time (Entprellung)
Werte : 0..5000 [ms]

ADC Eingänge
Werte : 0..255

PWM Ausgänge
Wertte : 0..255  (0 = 0 % ; 255 = 100%)
