TEMPORIZADOR CONTADOR

 

Este bloque de función genera impulos de duración controlada pudiéndose gobernar de acuerdo a las señales siguientes:

SEÑALES DE ENTRADA:

play = Activa la cuenta cuando es "1" y la para cuando es "0"

pause = Detiene el contador

reset = Pone a cero el contador

La salida del contador es la señal "counter" de tipo entero (integer).

El parametro de duracion de los impulso de cuenta es una variable de entrada de tipo integer etiquetada como "impulse"
