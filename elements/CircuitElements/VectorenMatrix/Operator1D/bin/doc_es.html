Función: OPERADOR BASICO 1D

Esta función permite a su entrada dos valores numéricos A y B de la misma o distinta naturaleza y realiza sobre ellos las operaciones:

  • ADD Suma
  • SUB Resta
  • MUL Multiplicacion
  • DIV Division

La posiblidad de que los valores numericos de entrada sean distinta naturaleza le hace muy úti. Sin embargo es preciso tener en cuenta que la salida del resultado sera de acuerdo a la tabla adjunta

Entrada A Entrada B Salida
INT INT INT
INT DBL DBL
DBL INT DBL
DB L DBL DBL
1D-INT 1D-INT 1D-INT
1D-INT INT 1D-INT
INT 1D-INT 1D-INT
1D-DBL 1D-DBL 1D-DBL
1D-DBL DBL 1D-DBL
DBL 1D-DBL 1D-DBL
