ELEMENT NAME:  "Append 2D String To Textfile"

DEVELOPED BY: Carmelo Salafia  EMAIL: cswi@gmx.de COUNTRY: Germany

DESCRIPTION:

This element appends a 2D string to a file of text or CSV by specifying the location of the file in the input Patch.

The specified file is automatically created if the specified file does not exist, and the path is invalid.

BASIC EXAMPLE:

You can find this example at Projects window of MyOpenLab >  "VirtualMachines>English Examples>VectorMatrix > Append To File"
