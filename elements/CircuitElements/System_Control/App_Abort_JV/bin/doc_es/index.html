NOMBRE DEL ELEMENTO: App_Abort_JV

DESCRIPCIÓN:

Este elemento permite cerrar la máquina virtual de Java (JVM) para cerrar la aplicación programáticamente.

EJEMPLO BÁSICO DE FUNCIONAMIENTO:

FRONT ELEMENTS:

CODE:

Puede encontrar el ejemplo en la carpeta de Ejemplos, ubicada en la ventana de "Proyectos>VirtualMachines> Espanol Ejemplos>Control de Aplicación>Abort_App_JV_Example".

 

ENTRADAS:

1, " Abort_Application" (Boolean):

Si es True ejecuta el cierre de la máquina virtual de Java (JVM), si es false no hace nada y la aplicación sigue ejecutándose.
