Exte dispositivo es muy úti para recoger datos y gradarlos en un fichero.

Las señales de entrada son:

Cargar: Carga Dato

Nombre del Fichero: Patch y nombre del fichero sobre el que escribiremos (debe estar previamente creado)

Canal1: Canal 1 de entrada de datos de tipo dbl

Canal2: Canal 2 de entrada de datos de tipo dbl

Las salidas de datos:

Salida de Datos: Salida de los datos en una tabla 2D (str)
