Start Element

The Start Element is necessary for starting the flowchart Elements and it have to be at the beginning of every flowchart.
The Start Element represents a Procedure or Function, and he is the owner of all appended Flowchart Elements.

Parameter :
You can define Parameters :

 Datatypes    :

Double
String
Boolean
