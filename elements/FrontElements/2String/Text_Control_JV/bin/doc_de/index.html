DESCRIPCIÓN:  This Element allows show multiline text and enables scroll bars when needed. (Mouse scroll enabled).

The ekement allows configure the next properties for customize the element appearance:

 "Font",  "Init_Text", "Font_color", "Text_Align" and "BackGround Color".

Aditional Information:

" Text_Align ":

0=LEFT

1= CENTER

2=RIGHT

3=JUSTIFIED
