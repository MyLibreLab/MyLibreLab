Este componente emula el movimiento de un cilindro neumático.

Tiene las siguientes Entradas:

ON: Activa el cilindro
OFF: Dessactiva el cilindro
Manual-Automatico: Pone el cilindro en modo manual o en modo automático
Posición: Posiciona el cilindro en una posición siempre que este en modo manual

Este es un montaje característico.
