Objetos de Entrada (Botones/Interuptores): BOTON

Este elemento actua generando una variable Booleana en funcion de si esta pulsado o no.

Atributos modificables:
Texto Permite poner un texto al botón
Color del Texto Permite poner color al texto
Color de Fondo Perimte cambiar el color del fondo del botón
Interruptor Se puede gacer que el boton actue como mun interruptor
Init-Valor Perite que al inicar la simulacion la variable asociada al interrutor tenga un valor
