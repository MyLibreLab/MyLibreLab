Embedded-VM Element

Dieses Element ist ähnlich einer "Function/Procedure" und kapselt eine 'komplette VM 
(inkl. Circuit und Front-Panel) in einem Element, wobei das Front-Panel der zu ladenden 
VM im  Front-Panel eingebettet wird.

Als  Schnittstellen werden Input/Output Pins verwendet, die in der zu ladenden VM
eingefügt werden müssen. Diese werden dann als I/O Pins im Element dargestellt.

!Vorsicht : beim späteren entfernen oder hinzufügen von  Pins, kann es passieren das
das Element neu Verdrahtet werden muss.

Beim hinzufügen dieses Element in das Circuit-Panel ist zu beachten das der
Relative Pfad zur VM inkl. der Dateiendung ".vlogic" angegeben werden muss.

Parameter : keine
