Objetos de Salida de Información/Datos: OSCILOSCOPIO MULTICAL (5 Canales)

Este elemento permite el trazado gráfico de hasta 5 señales que varian en el tiempo

ELEMENTOS QUE SE PUEDEN MODIFICAR

  • El color con el que dibuja la señal. (color canal)
  • El fondo del Osciloscopio

En el ejemplo se ve una aplicacion de un osciloscopio multicanal alimentado por dos señales.

Observar que las señales se han colocado con distintos ofssets para que no se solapen.

El rayado es un BMP que se coloca y se lleva al fondo y a la vez se deshabilta la opción Mostrar Fondo con el fin de que el fondo sea el rayado.
