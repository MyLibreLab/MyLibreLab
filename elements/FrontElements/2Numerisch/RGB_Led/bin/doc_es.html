Objetos de Salida de Información/Datos: LED Multicolor (RGB)

Este elemento muestra un diodo led en el que se puede seleccionar el color mediante la mezcla de los tres colores básicos Rojo, Azul y Verde

En el ejemplo se ve un diodo al cual le podemos modificar el color basta con mover el potenciometro del color correspondiente.

Importante: El valor del color estará comprendido entre 0 y 255 (si es mayor de 255 el programa da un error)
