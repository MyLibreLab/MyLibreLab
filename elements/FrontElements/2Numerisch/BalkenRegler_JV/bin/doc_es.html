Objetos de Entrada de Información/Datos: SLIDER (BARRA DE CONTROL DESLIZANTE)

Este elemento genera una variable de valor analógico en un rango modificable por el usuario.

ELEMENTOS QUE SE PUEDEN MODIFICAR

Valor Min:. Valor mínimo que puede dar el control
Valor Max: Valor máximo que puede dar el control
Valor Inicial: Valor que adopta el control cuando se inica la simulación
Color Fondo: El cvolor del fondo del control
Color Contorno: Color del cursor que se desliza
Marcas: Numero de divisiones del control
