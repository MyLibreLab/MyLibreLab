Objetos de Salida de Información/Datos: BARRA GRAFICA

 

Este elemento permite dibujar una barra de color en fucnion del valor de la variable de se entrada

La variable de Entrada es de tipo Entero (dbl).

 

ELEMENTOS QUE SE PUEDEN MODIFICAR

Valor Min:. Valor mínimo que se indicará
Valor Max: Valor máximo que se indicará
Color Mezcla A: Modifica el color A utilizado en la barra
Color Mezcla B: Modifica el color B utilizado en la barra
Color Letra: Color de la letra
Marcas: Numero de divisiones de la barra de color
