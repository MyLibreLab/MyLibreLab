Objetos de Salida de Información/Datos: REGISTRADOR DE DATOS

 

Este elemento permite realizar el trazado en una pantalla grafica de datos.

La variable de Entrada es de tipo Entero (dbl).

La señal Reset permite limpiar la pantalla

PARAMETROS EDITABLES DE ESTE OBJETO

Mostrar Nombre: Permite mostrar el nombre del objeto
Mostrar Rejilla: Permite mostrar y ocultar la rejilla
Mostrar Fondo Permite mostrar y ocultar el fond
Tiempo de Refresco: Perite modificar el tiempo de scaneado
